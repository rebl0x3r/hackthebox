<h2 align="center">Installation</h2>

  HackTheBox requires <a href="https://www.python.org/">Python </a> v3 to run.
  
  <h3>Linux</h3>
  
  ```
  $ git clone https://github.com/rebl0x3r/hackthebox.git
  
  $ cd hackthebox

  $ chmod +x main.py
  
  $ pip3 install -r requirements.txt

  $ ./main.py
  ```
